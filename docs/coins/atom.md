# Cosmos

This is a Cosmos Driver for the Platform SDK. The implementation makes use of [Cosmos](https://cosmos.network/) and adheres to the contracts laid out in the [specification](/docs/specification.md). **This package should not be used directly. It is designed to be consumed through the [SDK](/docs/sdk.md) package.**

## Installation

```bash
yarn add @payvo/sdk-atom
```

## Security

If you discover a security vulnerability within this package, please send an e-mail to [security@payvo.com](mailto:security@payvo.com). All security vulnerabilities will be promptly addressed.
